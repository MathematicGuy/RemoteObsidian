Test - **A number calculated by a statistical test that describes how far the observed data is from the null hypothesis.**

| **Data Type**   | **Definition**                                         | **Mathematical Operations**        | **Examples**                  |
| --------------- | ------------------------------------------------------ | ---------------------------------- | ----------------------------- |
| **Nominal**     | Categorical data with no inherent order or ranking.    | Counting (Frequency), Mode         | Gender, Color, Country        |
| **Ordinal**     | Categorical data with a clear, logical order or rank.  | Comparison ($>$ or $<$), Median    | Education level, Star ratings |
| **Categorical** | An umbrella term for Nominal and Ordinal data.         | Varies by sub-type                 | Occupation, Risk level        |
| **Continuous**  | Numerical data that can take any value within a range. | Mean, Standard Deviation, Calculus | Height, Temperature, Time     |

|**Variable Type**|**Synonyms**|**Role in Research**|
|---|---|---|
|**Explanatory**|Independent ($X$), Predictor|The variable you manipulate or use to predict a change.|
|**Response**|Dependent ($Y$), Outcome|The variable that is measured; it is expected to change based on the explanatory variable.|
|**Control**|Constant, Nuisance|A variable kept the same throughout the experiment to ensure a fair test.|
|**Confounding**|Lurking variable, Extraneous|An outside influence that changes both the explanatory and response variables, potentially invalidating results.|
