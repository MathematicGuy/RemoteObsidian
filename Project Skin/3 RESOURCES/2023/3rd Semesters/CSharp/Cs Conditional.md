### If Else
```cs
if (condition) {
	// code
}
else if (more condition){
	// code
}
else if (even more condition){
	// code
} 
```

### Switch
```cs
num = 2
switch (num) {
	case 1:
		Console.WriteLine("One");
		break;
	case 2:
		Console.WriteLine("Two");
		break;
	case n:
		Console.WriteLine("n");
		break;
	default:
		// Like Else
		Console.WriteLine("default")
		break;
}
>> "Two"
```
